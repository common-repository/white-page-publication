== Description ==
*WhitePage* est un plugin très léger qui permet à la plateform white.page de publier facilement sur votre site web Wordpress. Il intercepte simplement les demandes de articles et fonctionne sur n'importe quel site WordPress. 

= Français =

*WhitePage* est un plugin très léger qui permet à la plateform white.page de publier facilement sur votre site web Wordpress. Il intercepte simplement les demandes de articles et fonctionne sur n'importe quel site WordPress. 

Ce plugin vous est gentiment proposé par <a href="https://www.white.page/white-page-publication.zip" target="_blank">white.page</a>.


Ce plugin est seulement maintenu, ce qui signifie que nous ne garantissons pas un support gratuit. Envisagez de signaler un problème et soyez patient.

= Compatibility =

= Français =

Nécessite WordPress 4.1 ou supérieur. Toutes les choses liées à la création des artices.

== Installation == 

= Français =

1. Aller dans Extensions › Ajouter.
2. Rechercher *WhitePage*.
3. Recherchez ce plugin, téléchargez-le et activez-le.
4. Acceder au menu White Page afin d'enregistrer vos clés api pour la configuration Api.

== FAQ ==

= Comment obtenir la clé privée et public pour la configuration ! =

Allez sur le site web www.white.page et créer un compte client. Une fois connectez à votre espace sur white.page, il faudra ajouter un site (site sur lequel vous comptez installer le plugin). Vous disposez automatique des clés api une fois le site ajouter avec succès...

Recopiez les clés afin de finaliser la configuration du plugin sur votre site web...
